## ❓ Troubleshooting

If you encounter issues during installation or usage, try these steps:

* Installation Fails: Check if your antivirus software prevents the file from running. Disable it temporarily and try the installer again.
* No Suggestions Appear: Ensure the application is running in your system tray. Look for the Grammarly logo near the clock on your taskbar.
* Application Crashes: Restart the app by closing it from the taskbar and launching it from the Start menu.
* Connection Errors: Verify that your internet connection is active. The AI tools require network access to analyze your text.
* App Compatibility: If the tool does not work inside a specific program, try restarting that program. Most applications receive support automatically.
